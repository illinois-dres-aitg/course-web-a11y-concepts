Disclosure Navigation Menu with Top-Level Links

The following example demonstrates using the disclosure design pattern to show and hide dropdown lists of links in a navigation bar for a mythical university web site. Each disclosure button opens and closes a menu of links. This example includes top-level links in the navigation menu in addition to the buttons that show and hide a list of links.

Keyboard Support

Key Optional Funtion
Tab Move to next disclosure button, top-level link or menu link.
Shift+Tab Move to previous disclosure button, top-level link or menu link.
Space and
Enter		 Open or close a menu.
Esc Closes an open menu.
Up Arrow Yes Opens menu or moves to previous link in an open menu.
Down Arrow Yes Opens menu or moves to next link in an open menu.
Left Arrow Yes Moves to previous top level item, closes menu if open.
Right Arrow Yes Moves to next top level item, closes menu if open.
Home Yes Moves to first top level item or first link in an open menu.
End Yes Moves to last top level item or last link in an open menu.
